# rainbow-zsh
A simple ZSH theme which introduces a rainbow-gradient prompt.

## Installation:
1. Download rainbow.zsh-theme.
2. Put rainbow.zsh-theme in ~/.oh-my-zsh/themes/
3. Configure ~/.zshrc `ZSH_THEME="xxx"` to `ZSH_THEME="rainbow"`.\
